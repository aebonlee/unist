# 강의안 슬라이드(PPT) 제작 — 2026-06-22

## 개요
"데이터 분석 실습 & 시각화"(KERIS 공통직무 · 1일 6시간 중급) 과정의 **강의안 프레젠테이션**을 제작했다.
별도 라이브러리·빌드 없이 동작하는 **단일 HTML 슬라이드 덱**(`public/slides.html`)으로 만들어,
배포 시 `https://data.dreamitbiz.com/slides.html` 로 바로 열린다.

## 산출물
- `public/slides.html` — 27장 슬라이드 덱 (vanilla JS, 의존성 0)
  - 표지 → 과정 개요 → 아젠다 → 모듈 1/2/3(각 디바이더 + 본문) → 회고 → 클로징
  - 모든 콘텐츠는 실제 강의안(`src/pages/Module1/2/3.tsx`)에서 발췌해 정합성 유지
- `src/pages/Lecture.tsx` — 강의안 허브 상단에 "📊 강의안 슬라이드(PPT) 열기" 링크 추가
  (`import.meta.env.BASE_URL` 기반, 새 탭)

## 디자인
- 사이트 디자인 시스템과 통일: 다크 블루 navy(`#1B2A4A`) 배경 + 블루 액센트
- Pretendard 폰트, 카드/표/스텝/콜아웃/공식 박스 등 슬라이드 컴포넌트
- `clamp()` 반응형 타이포 → 빔프로젝터·노트북·모바일 모두 대응

## 기능
- 키보드: ← → / Space / PageUp·Down / Home·End 이동, **F** 전체화면, **P** 인쇄(PDF 저장)
- 상단 진행바, 우하단 카운터(n/총), 좌하단 이전/다음 버튼
- 모바일 스와이프 좌우 네비게이션
- `location.hash` 동기화 → 특정 슬라이드 URL 공유 가능 (예: `slides.html#12`)
- `@media print` — 1280×720 페이지로 PDF 내보내기 시 슬라이드별 페이지 분할

## 추가 — 편집 가능한 PowerPoint 원본(.pptx) (2026-06-22)

웹 슬라이드와 별도로, **편집 가능한 네이티브 PPTX 원본**을 생성했다.

### 산출물
- `scripts/generate-pptx.py` — python-pptx 기반 생성 스크립트 (28장)
- `public/강의안_데이터분석_시각화.pptx` — 16:9, navy 테마, slides.html과 동일 구성
- `package.json` 에 `npm run pptx` 스크립트 추가
- `Lecture.tsx` 에 "⬇ PPT 원본(.pptx) 내려받기" 링크 추가

### 특징
- 16:9(13.33"×7.5") · 다크 블루(#1B2A4A) 배경 + 블루 액센트로 웹 덱과 통일
- 한글 폰트 `맑은 고딕`(EastAsian typeface 명시) — PowerPoint에서 바로 렌더
- 표·카드·콜아웃·공식 박스·LAB 태그·진행바·페이지 카운터까지 재현
- 모든 텍스트/표가 **편집 가능한 네이티브 개체** (이미지 아님)

### 재생성
```bash
npm run pptx          # python3 scripts/generate-pptx.py
```
> 의존성: `pip3 install python-pptx` (개발 머신엔 1.0.2 설치됨)

### 사용법
- 다운로드: 강의안 페이지의 "PPT 원본(.pptx) 내려받기" 또는
  `data.dreamitbiz.com/강의안_데이터분석_시각화.pptx`
- PowerPoint/Keynote/Google Slides에서 열어 자유롭게 편집
