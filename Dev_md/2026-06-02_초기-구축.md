# 개발 로그 — 데이터 분석 실습 & 시각화 사이트 초기 구축

- **일자**: 2026-06-02
- **목표**: `aebonlee/data` 저장소를 데이터 전처리·분석·시각화 강의용 사이트로 구축
- **배포 도메인**: data.dreamitbiz.com

## 작업 요약

### 1. 스캐폴드
- `dasco`(GitHub) 디자인 템플릿을 클론하여 `data` 저장소에 복사 (React 19 + TS + Vite 7 + Supabase).
- dasco/kdn 은 동일 디자인 시스템(다크 블루 navy 기본 + 5색 테마, `src/styles/*.css`).
- 루트 레거시 파일(app.jsx, tweaks-panel.jsx, styles.css 등) 및 미사용 페이지 제거.

### 2. 브랜딩 전환 (`src/config/site.ts`, `index.html`)
- id `dasco`→`data`, name `Data Lab`, dbPrefix `data_`, url `data.dreamitbiz.com`.
- 메뉴: 과정 소개 / 커리큘럼 / 강의안(모듈1·2·3) / 추천 자료.
- CNAME, robots.txt, sitemap.xml, OG/meta 태그를 data 도메인으로 갱신.
- Navbar·Footer 하드코딩 브랜드 `Dasco & AI` → `Data Lab` 로 교체.
- 기본 컬러 **다크 블루(`#1B2A4A`)** + 5색 테마(blue/red/green/purple/orange) 유지.

### 3. 인증 (Supabase)
- `.env` 에 `VITE_SUPABASE_URL` / `VITE_SUPABASE_ANON_KEY` 설정 (`VITE_` 접두사, gitignore됨).
- Google·Kakao OAuth 는 `src/utils/auth.ts` 에 이미 배선됨 — Supabase 대시보드 Provider 활성화만 필요.
- service-role/PAT(`sbp_...`)는 클라이언트에 미포함 (관리 전용으로 분리).

### 4. 콘텐츠 (KERIS 세부 커리큘럼 반영)
- `Home.tsx`: 6시간·3모듈 랜딩 (히어로/모듈/타임라인/도구/원칙/CTA).
- `Curriculum.tsx`: 교육 목표 + 3개 모듈 세부 내용(강의/실습 구분).
- `Lecture.tsx`: 모듈 허브. `Module1/2/3.tsx`: GuidePage 기반 마크다운 강의안.
  - 모듈1 EDA(7섹션), 모듈2 시각화 원리(6섹션), 모듈3 대시보드(6섹션).
- `About`, `RecommendedSites`(데이터 출처·시각화 도구·학습자료)로 교체.
- 라우트 정리: `/curriculum`, `/lecture`, `/lecture/module1~3`.

### 5. OG 이미지 + 검증
- `scripts/generate-og.cjs` 를 데이터 강의/다크블루(막대차트 모티프)로 수정, sharp 로 `public/og-image.png`(1200×630) 생성.
- `npm install` → `npm run build` 성공 (tsc 타입체크 통과, 414 모듈), `npm run preview` 구동 확인.
