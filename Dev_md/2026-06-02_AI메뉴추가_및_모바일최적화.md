## 1. Supabase 인증 DB
- `scripts/create-user-profiles.sql` — 공용 `user_profiles`(컬럼+RLS+가입 트리거). 적용 완료.

- Google·Kakao OAuth는 `src/utils/auth.ts`에 배선됨 — Supabase 대시보드 Provider 활성화 필요.

## 2. 배포 (GitHub Pages)
- 초기 빈 페이지 원인: Pages가 `main`/루트(`/src/main.tsx` 개발 진입점) 서빙 → 빈 화면.
- 해결: `dist`를 **`gh-pages` 브랜치**로 배포(`npm run deploy`, gh-pages devDep) + Pages 소스를 gh-pages로 전환(API).
- 빌드 도구: nvm node v22.22.2 (`export PATH="$HOME/.nvm/versions/node/v22.22.2/bin:$PATH"`).

## 5. 모바일 최적화 (Playwright 390/360px 실측)
- **GuidePage**(모듈/학습자료/프롬프트학습): 모바일 사이드바 TOC를 본문 위로 세로 스택(@900), 마크다운 표는 컨테이너 내 가로 스크롤.
- **aifree.css**: `@media` 블록을 base 규칙 뒤로 이동(순서 버그로 PromptPractice 안 깨지던 문제) + 600px 보강.
- **드롭다운**: 모바일 햄버거에서 `width:auto`로 전체 폭 복원.
- **About/CompanyIntro**: 인라인 고정 그리드(`240px 1fr`, `repeat(3,1fr)`, `1fr 1fr`)를 className(`instr-*`, `co-2col`)+미디어쿼리로 모바일 스택.
- 결과: 전 페이지 가로 오버플로우 0px (실측 검증).
