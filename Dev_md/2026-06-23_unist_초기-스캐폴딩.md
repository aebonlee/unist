# 2026-06-23 · UNIST 사이트 초기 스캐폴딩

## 개요
`data` 프로젝트(data.dreamitbiz.com, KERIS 데이터 분석 과정)를 기반으로 **UNIST 학습 사이트**(unist.dreamitbiz.com) 초기 구축.

## 작업 내용
- `aebonlee/data` 작업 트리를 `aebonlee/unist`로 복사 (`.git`/`node_modules`/`dist` 제외)
- 리브랜딩
  - `package.json` — `name: unist`, `homepage: https://unist.dreamitbiz.com`
  - `src/config/site.ts` — `id: unist`, `url`, `dbPrefix: unist_`, 브랜드 `UN`+`IST`, 헤더 주석
  - `.env.example` — `VITE_SITE_URL`
  - 루트/`public` `CNAME` → `unist.dreamitbiz.com`
  - `README.md` 재작성
- 강의 원자료 폴더 신설: `data/` (`data/ppt`, `data/excel`, README)
- `npm install` + `npm run build` 통과 확인 (Vite 빌드 정상)
