# 강의 원자료 (UNIST)

이 폴더는 UNIST 강의 사이트 제작에 사용하는 **원본 자료**를 보관합니다.
여기에 자료를 넣어주시면, 그 내용을 바탕으로 커리큘럼·강의안·실습 페이지를 작성/갱신합니다.

## 받을 자료

- [ ] 작성된 PPT 1 — `(파일명)`
- [ ] 작성된 PPT 2 — `(파일명)`
- [ ] 엑셀 문제 1 — `(파일명)`

## 파일 배치 가이드

```
data/
  ppt/        # 강의 슬라이드 (.pptx, .pdf)
  excel/      # 엑셀 실습 문제 (.xlsx)
  README.md   # 이 파일
```

> 자료를 추가한 뒤 알려주시면, PPT 흐름에 맞춰 강의안(Module 페이지)을,
> 엑셀 문제에 맞춰 실습/예제 페이지를 구성합니다.
